# Empty React App
An empty React app written in TypeScript with Webpack and Babel baked in for easy development.
## Pre-requsites
- Node JS
- Yarn

## Installation
```
yarn install
```

## Development Server
```
yarn start
```

## Build
```
yarn build
```
